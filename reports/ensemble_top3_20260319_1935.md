# 앙상블 실험: Top 3 모델 투표

## 구성

- **LGBM_base**: 강세장 최강 (+58.68%)
- **XGB_base**: 약세장 최강 (+5.77%), 낮은 MDD
- **CatB_tuned**: 강세장 2위 (+42.55%), 낮은 MDD

### 앙상블 방식 3가지

| 방식 | 설명 |
|---|---|
| Hard Voting | 3개 모델 다수결 (2/3 이상 동의) |
| Soft Voting | 3개 모델 확률 평균 → argmax |
| Conservative | Buy=만장일치, Sell=2/3이상, 나머지=Hold |

## 분석

### 1. 앙상블이 최강 단독 모델을 이기지 못함

- 강세장: LGBM_base(+58.68%) > 모든 앙상블(최대 +36.46%)
- 약세장: XGB_base(+5.77%) > 모든 앙상블(최대 -2.46%)

앙상블은 최악 모델의 나쁜 판단이 다수결에 섞여 최강 모델의 정확한 판단을 희석시킴.

### 2. Conservative 앙상블이 가장 나은 앙상블 방식

- 거래 수가 극도로 적음 (56회, 18회) — 만장일치 Buy 조건이 매우 까다로움
- 강세장: +36.46% (Sharpe 1.68, MDD -12.65%) — 수익률은 낮지만 리스크 대비 효율적
- 약세장: -2.63% — B&H(-6.51%)보다는 나음

### 3. Soft Voting이 가장 나쁨

확률 평균은 3개 모델의 불확실성을 모두 흡수하여 오히려 애매한 판단 증가. 약세장에서 MDD -26.47%로 최악.

### 4. 왜 앙상블이 실패하는가

3개 모델이 모두 **같은 regime 분류** 패러다임이기 때문:
- 변동성 기반으로 시장 상태를 판단하는 동일한 메커니즘
- 서로 다른 알고리즘이지만, 학습하는 패턴이 유사
- "다양성"이 부족하여 앙상블의 핵심 이점(error decorrelation)이 작동하지 않음

진정한 앙상블 효과를 위해서는 **다른 피처 세트, 다른 시간 프레임, 다른 라벨링 방식**을 사용하는 모델이 필요.

## 결론

- 현재 Top 3 앙상블은 **LGBM_base 단독보다 열등**
- Conservative 방식은 낮은 거래 빈도와 MDD 억제에는 효과적
- 같은 피처 + 같은 라벨로 학습한 GBDT 3종의 앙상블은 다양성 부족으로 의미 없음
- **LGBM_base를 메인으로 유지**하되, 개선 방향은 모델 교체가 아닌 피처/라벨/프레임워크 혁신이 필요
